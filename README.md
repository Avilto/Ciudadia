# 🏠 Ciudadia

**Plataforma de Oferta de Inmuebles**

Ciudadia es una aplicación web moderna para la búsqueda y oferta de inmuebles. Diseñada con un enfoque premium, ofrece una experiencia visual impactante con tema oscuro, glassmorphism y micro-animaciones.

## ✨ Características

- 🔍 **Búsqueda inteligente** — Filtra por tipo de inmueble, precio y ubicación
- 🏡 **Catálogo variado** — Casas, departamentos, terrenos y locales comerciales
- 💎 **Diseño premium** — Dark mode con acentos dorados y efectos glassmorphism
- 📱 **Responsive** — Adaptado para desktop, tablet y móvil
- ⚡ **Rendimiento** — HTML/CSS/JS puro, sin frameworks pesados
- 🎯 **Interactivo** — Modales de detalle, favoritos y filtros dinámicos

## 🚀 Uso

Simplemente abre `index.html` en tu navegador:

```bash
# O usa un servidor local
npx serve .
```

## 🛠️ Tecnologías

- **HTML5** — Estructura semántica
- **CSS3** — Custom properties, Grid, Flexbox, animaciones
- **JavaScript** — Vanilla JS (ES6+)
- **Google Fonts** — Outfit + Inter
